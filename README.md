# Student Result Processing System



## Overview
The **Student Result Processing System** is a Java-based console application designed to manage student records efficiently.
It automates the process of storing, updating, and retrieving student marks, calculating totals, averages, and grades. 
This project demonstrates **object-oriented programming**, **file handling**, **exception handling**, and **menu-driven programming** in Java.

---

## Features
- Add, edit, and delete student records.
- Automatically calculate total marks, average, and grade.
- Search for students by ID or name.
- Identify failed students easily.
- Persistent storage of records using files.
- User-friendly menu-driven interface.

---

## Technologies Used
- **Programming Language:** Java  
- **Platform:** Console-based  
- **File Handling:** Sequential file storage for student data  
- **IDE:** Any Java IDE (Eclipse, IntelliJ IDEA, NetBeans)
